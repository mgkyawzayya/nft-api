### Setup

```
$ git clone https://github.com/mgkyawzayya/nft-api.git

$ cd nft-api

$ npm install && npm run dev

$ composer install && cp .env.example .env && php artisan key:generate

$ php artisan migrate --seed

$ php artisan serve

```
